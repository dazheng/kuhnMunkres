# KunhnMunkress
带权二分图最大匹配Kuhn-Munkres算法go语言实现。是将 http://csclab.murraystate.edu/~bob.pilgrim/445/munkres.html 中的c#实现改为go实现，同时参考了https://github.com/zarrabeitia/munkres_sparse

## Installation

```
go get -u github.com/dazheng/kuhnmunkres
```
